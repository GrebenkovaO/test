EDGS: Eliminating Densification for Efficient Convergence of 3DGS

Dmytro Kotovenko*, Olga Grebenkova*, Björn Ommer

CompVis @ LMU Munich • Munich Center for Machine Learning (MCML)

*Equal contribution

Paper Code Demo

EDGS replaces the incremental refinement process to speed up convergence — enabling high-quality and fast 3D reconstuction.

Abstract

3D Gaussian Splatting (3DGS) reconstructs scenes starting from a sparse SfM initialization and iteratively refines under-reconstructed regions. This is slow due to repeated densification — splitting Gaussians and optimizing them through a lengthy process.

We propose a fundamentally different approach: initialize 3D Gaussians directly using triangulated pixels from dense image correspondences. This eliminates densification, reduces optimization time, and improves consistency across high-frequency regions.

EDGS optimizes all splats in parallel from the beginning, significantly accelerating convergence and improving rendering quality — all while using half as many splats as standard 3DGS. It's compatible with existing 3DGS acceleration methods.
